# 🎲 Dice Rolling Simulator

A simple command-line Dice Rolling Simulator built with Python that mimics the behavior of a real six-sided die. The program generates random dice rolls and displays visually formatted dice faces using Unicode dots.

## 🚀 Features

* Roll a virtual six-sided die
* Choose how many times to roll
* Random outcome generation using Python's `random` module
* Visual dice face rendering
* Input validation for a smooth user experience
* Beginner-friendly and easy to understand

## ▶️ How to Run

1. Clone this repository

```bash
git clone <https://github.com/mirunalinibuilds/dice-rolling-simulator>
```

2. Navigate to the project directory

```bash
cd dice-rolling-simulator
```

3. Run the program

```bash
python main.py
```
