# 🎲 Roll A Die - Python Dice Rolling Simulator

A beginner-friendly Python project that simulates a real six-sided die using random number generation. The program allows users to roll the die multiple times and displays visually formatted dice faces using Unicode dots.

This project was built as part of my Python learning journey to strengthen programming fundamentals through hands-on practice.

## 🚀 Features

* Simulates a six-sided die
* Generates random outcomes using Python's `random` module
* Allows multiple dice rolls in a single session
* Displays realistic dice faces using Unicode dots
* Handles user input and validation
* Simple and beginner-friendly implementation
